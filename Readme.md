## Restore

* Clone this repo to your home directory
* Run the `setup.sh` file

```sh
$ sh setup.sh
```

> Note: This will automatically create a `dotfiles_old` folder and backup any exsisting files.


## Get VIM Ready

```sh
$ vim # Any file will do
:BundleInstall
```
